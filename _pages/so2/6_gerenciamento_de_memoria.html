5. Gerenciamento de Memória

Introducao

Espaço de endereçamento: text -> rodata -> bss (variáveis estáticas) -> heap -> stack -> share libs -> SO.

Localidade temporal e espacial

O endenreço formado pelo CS e o desolocamento precida ser trduzido para um endereço virtual base.

Tradução de endereço

Os ajustes de áreas de memória -> execve

Arquivo ELF -> dividido em segmentos

Utilitários: readelf, ...

ELF

Loader, responsável por carregar o código de um aquivo executável.

Hexdump

Readelf: ponto de entrada é o endereço da main

Malloc e Free

O espaço lógico de um prcesso.

No task structu exite o mm, que mapeia as áreas de memória

  • .rodata
  • .bss
  • .heap: alocação dinâmica.
  • .stack
  • shared libs
  • SO
